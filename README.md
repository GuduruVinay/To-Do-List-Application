# To-Do List Application (React + Vite + Tailwind CSS)

A simple and clean To-Do List Application built using React, Vite, and Tailwind CSS. The app allows users to add, edit, delete, filter, and mark tasks as completed.
Dark mode is also supported and saved in localStorage.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed / uncompleted
- Filter tasks (All / Active / Completed)
- Clear completed tasks
- Progress bar indicator

## UI & UX

- Fully responsive interface
- Dark mode toggle (saved in localStorage)
- Tailwind CSS styling
- Smooth transitions

## Local Storage

- Tasks persist even after reloading
- Dark mode theme persists

## Technologies Used

- React 18
- Vite
- Tailwind CSS v4
- ESLint
- LocalStorage API

## Installation & Running the Project

Follow these steps to run the application on your computer :

### Clone the Repository

```bash
git clone https://github.com/GuduruVinay/To-Do-List-Application.git
```

Or download the ZIP manually and extract it.

### Install Dependencies

Open the project folder in your terminal :

```bash
cd To-Do-List-Application
npm install
```

This installs React, Vite, Tailwind CSS, and required dependencies.

### Run the Development Server

```bash
npm run dev
```

Then open the local server link shown in the terminal, for example :

```bash
http://localhost:5173/
```

The To-Do List Application will now be running !
